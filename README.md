# 漢字クイズ

小学生向けの漢字学習PWA（Progressive Web App）です。

## 機能

- 🎯 1〜6年生の学年別漢字クイズ
- 📖 読みテスト・✏️ 書きテスト
- 📱 PWA対応（ホーム画面追加可能）
- 🔄 オフライン対応
- 📝 学年に応じた漢字表記調整

## 使用方法

### ブラウザで使用
https://[ユーザー名].github.io/kanji-quiz

### PWAとして使用（推奨）
1. 上記URLにアクセス
2. Safari: 共有ボタン → 「ホーム画面に追加」
3. Chrome: メニュー → 「ホーム画面に追加」

## 技術仕様

- HTML5 + CSS3 + JavaScript
- Service Worker（オフライン対応）
- Web App Manifest（PWA対応）
- レスポンシブデザイン
